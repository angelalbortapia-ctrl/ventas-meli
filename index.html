☁️ Sync Supabase (Mac ↔ iPhone)

Recomendado: abre esta PWA en Safari → Compartir → Añadir a pantalla de inicio. No hace falta App Store. Necesitas hospedar la app en HTTPS (GitHub Pages, Netlify, etc.) y el mismo login en ambos dispositivos.

  1. Crea un proyecto en supabase.com
  2. SQL Editor → pega y ejecuta supabase/schema.sql
  3. Project Settings → API → copia URL y anon key aquí
  4. Crea cuenta / inicia sesión con el mismo email en Mac e iPhone
  5. iPhone: abre la URL publicada (GitHub Pages) en Safari → Compartir → Añadir a pantalla de inicio

URL iPhone (cuando Pages esté activo): https://angelalbortapia-ctrl.github.io/ventas-meli/
